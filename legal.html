Legal Statement

My website is my original work therefore, it is copyright protected. All images used on the website are licensed or copyright-free so that all permissions are respected and there are no infringements.

The General Data Protection Regulation (GDPR) was created in 2018 and regulates what the website owners can do with the data collected from the website. This only data collected in this website is through the contact form. I believe that asking for the name and e-mail of the user is proportional to the use of the form. The data collected will only be used in sending the form and will not be shared. The form is also optional, so every user is automatically opting out until they decide to fill out the form.

Security

This website is not publicly available, so the data collected in the contact form is very secure and not in any danger of being accessed by an unauthorised source. However, if I were to publish the website as a public domain, I would make sure the data is secure by avoiding the Open Web Application Security Project (OWASP) Top 10 security risks.
